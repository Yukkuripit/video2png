# 🎬 video2png

動画をアップロードするだけで、**最大50枚**の静止画（PNG）に自動で切り出すWebツールです。  
サムネイル作成や動画のダイジェスト確認にぴったり。ブラウザだけで完結するので、誰でもカンタンに使えます。

👉 **実際に使ってみる**：`https://[あなたのユーザー名].github.io/[リポジトリ名]/`

---

## ✨ video2png の特徴

- **動画をドラッグ＆ドロップするだけ**  
  MP4 / WebM / OGG など、お手持ちの動画ファイルをそのまま読み込めます。

- **かしこい自動抽帧**  
  動画の長さに応じてFPS（1秒あたりの枚数）を自動調整。  
  どんなに長い動画でも**最大50枚**にバランスよく収まるよう最適化します。

- **1枚ずつチェック＆保存**  
  切り出された画像は一覧表示。気に入ったカットだけを個別にダウンロードできます。

- **まとめて一括ダウンロード（ZIP）**  
  全フレームをワンクリックでZIPにまとめて保存。1枚ずつ保存する手間が省けます。

- **プライバシー完全保護**  
  動画データは**あなたのブラウザ上だけで処理**され、外部サーバーに送信されることは一切ありません。安心してご利用いただけます。

## 🖥️ 使い方（たったの3ステップ）

### 1️⃣ 動画をアップロード
画面上の枠内に動画ファイルを**ドラッグ＆ドロップ**するか、枠内をクリックしてファイルを選択します。

### 2️⃣ 「フレームを抽出」をクリック
自動で動画が分析され、画像の切り出しがスタート。進捗バーで状況がリアルタイムにわかります。

### 3️⃣ 画像をダウンロード
- 各画像の下にある「保存」ボタンで**個別に保存**
- 「すべてZIPで保存」ボタンで**まとめて一括ダウンロード**

お好みの方法で保存してください。

## ❓ よくある質問

**Q. 対応している動画形式は？**  
A. お使いのブラウザ（Chrome / Edge / Firefox / Safari など）が再生できる形式であれば、ほぼすべて対応しています。  
特に **MP4（H.264）** や **WebM** が安定して動作します。

**Q. 何枚抽出されるの？**  
A. 動画の長さに応じて自動で調整され、**最大50枚**を超えないようになっています。  
例えば50秒以上の動画は最低0.5FPS（2秒に1枚）まで下げて調整するため、長編動画でも50枚に収まります。

**Q. 1時間以上の長い動画でも大丈夫？**  
A. はい。その場合は約0.013FPS（約75秒に1枚）程度まで自動で間引かれ、最大50枚に収まるよう調整されます。

**Q. スマートフォンでも使える？**  
A. はい。スマホ・タブレットにも最適化されているので、どこでも手軽にお使いいただけます。

## 🔧 上級者向け：Python版（ローカル処理）

ブラウザ版に加えて、リポジトリには **Python + OpenCV** を使ったローカル実行スクリプト（`extract_frames.py`）も同梱しています。  
大量の動画をバッチ処理したい場合や、コマンドラインで素早く処理したい場合にご利用ください。

```bash
# インストール（初回のみ）
pip install opencv-python

# 実行例
python extract_frames.py あなたの動画.mp4 ./出力フォルダ
```
> ※ Python版はあくまで補助的な機能です。通常はブラウザ版だけで全て完結します。
